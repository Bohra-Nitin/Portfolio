xAPI Portfolio Analytics System

A recruiter-facing analytics-enabled portfolio demonstrating real-time xAPI tracking, engagement analytics, and Veracity Learning Record Store (LRS) integration.

Project Overview

This project demonstrates the implementation of xAPI-based behavioral analytics inside a recruiter-facing analytics-enabled portfolio. The system tracks portfolio engagement events, calculates time spent, and sends structured xAPI statements to Veracity Learning Record Store (LRS) using JavaScript and xAPIWrapper.
