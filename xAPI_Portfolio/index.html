xAPI Portfolio Analytics System

A recruiter-focused portfolio analytics system demonstrating real-time xAPI tracking, engagement analytics, and SCORM Cloud Learning Record Store integration.

Project Overview

This project demonstrates the implementation of xAPI-based behavioral analytics inside a recruiter-facing portfolio experience. The system tracks portfolio engagement events, calculates time spent, and sends structured xAPI statements to SCORM Cloud Learning Record Store (LRS) using JavaScript and xAPIWrapper.
